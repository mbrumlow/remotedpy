# RemoteDPY

A small remote desktp viewer using go, xlib, and html5.

http://youtu.be/MnatFWQxT1k
https://youtu.be/JbypF_ULZTs

# Required 

* libx11-dev
* libxdamage-dev
* libxcomposite-dev

>apt-get install libx11-dev libxdamage-dev libxcomposite-dev libxtst-dev

# Note
